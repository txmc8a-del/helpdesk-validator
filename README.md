# INSPIRE Validation Community
This space is used for discussions and announcements around INSPIRE validation & conformity testing. Before reporting a problem or making a pull request, please read the [contribution guidelines](contribution_guidelines.md). Also, please follow our [Code of Conduct](https://github.com/INSPIRE-MIF/helpdesk/blob/main/code-of-conduct.md) anytime when you interact with the INSPIRE community in this space.

**NOTE**: Following the [discontinuation of the central instance of the INSPIRE Reference Validator](https://knowledge-base.inspire.ec.europa.eu/news-and-publications/news/discontinuation-inspire-reference-validator-2026-04-01_en) on 1 April 2026, INSPIRE implementers and data providers are recommended to setup their own instance of the Validator. Hence, support in the helpdesk will only be provided for interpretation/clarification of test results obtained using your own instance of the INSPIRE Reference Validator. Instead, no support will be provided on deployment issues.

Use the [*Issue tracker*](https://github.com/INSPIRE-MIF/helpdesk-validator/issues/new/choose) to report a problem with the interpretation of results using your own instance of the INSPIRE Reference Validator.

Use the [*Discussions tab*](https://github.com/INSPIRE-MIF/helpdesk-validator/discussions/) to start a discussion or raise a question of interest to the community around the INSPIRE Reference Validator, choosing the appropriate category. 

Please **do not use the helpdesk to report security vulnerabilities** of the INSPIRE Reference Validator. To report these, please contact us at JRC-INSPIRE-SUPPORT AT ec.europa.eu.

Please note that the Issue tracker and the Discussions tab should not be used to report generic issues with INSPIRE implementation (for these, please use the [generic INSPIRE helpdesk](https://github.com/INSPIRE-MIF/helpdesk)), but only to report problems of the INSPIRE Reference Validator.

Please check the [changelog of the current and past releases](https://github.com/INSPIRE-MIF/helpdesk-validator/releases) to access the full set of releases of the INSPIRE Reference Validator.
